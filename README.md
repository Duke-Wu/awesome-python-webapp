# superBlogWebapp

superBlogWebapp 是由 aiomysql 与 aiohttp 编写的一个轻量架构的异步 Web框架。

它主要的特点是资源占用小，支持高并发， 支持简单的 ORM。


站点技术规划：

	[root@duke ~]# python3.5 --version
    Python 3.5.2
	
异步框架aiohttp

    $ pip3 install aiohttp
    
MySQL的Python异步驱动程序aiomysql：

    $ pip3 install aiomysql
	
前端模板引擎jinja2：

    $ pip3 install jinja2
